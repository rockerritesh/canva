Q1. Mean, Median, Mode and Correction Problem
Source: 2080-regular.pdf

Part A – Differences

  • Mean: Sum of all values divided by number of values (affected by outliers). Example: {2,4,6,8,10} → 6.
  • Median: Middle value when ordered (robust to outliers). Example: {2,4,6,8,10} → 6.
  • Mode: Most frequent value; can be none/one/many. Example: {2,3,4,4,5} → 4.

Part B – Correction

Given n=100, mean=40, SD=5; incorrect 53 replaced by 83.

  1. Original sum = 100×40 = 4000 → corrected sum = 4030 → new mean = 40.3.
  2. Original Σx² = n(SD² + mean²) = 100(25 + 1600) = 162,500.
  3. Corrected Σx² = 162,500 − 53² + 83² = 166,580.
  4. New SD = √[(166,580/100) − (40.3)²] = √41.71 = 6.46.

Q3. Binomial vs Negative Binomial
Source: 2080-regular.pdf
  • Binomial: fixed n, variable successes r. P(X=r)=C(n,r)p^r(1-p)^{n-r}.
  • Negative Binomial: fixed successes r, variable trials n. P(X=n)=C(n-1,r-1)p^r(1-p)^{n-r}.
  • Both: independent trials, constant p, discrete.
